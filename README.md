# 💖 Will You Go On A Date With Me?

A cute and interactive date request website built with HTML, CSS, and JavaScript. The project features a pastel pink aesthetic, an adorable bear GIF, animated hearts, confetti effects, and a playful "NO" button that runs away whenever someone tries to click it.

## ✨ Features

- 💖 Cute pink-themed design
- 🧸 Animated bear GIF
- 🌸 Responsive layout for desktop and mobile
- 😂 "NO" button that moves away from the cursor
- 🎉 Confetti celebration after clicking "YES"
- 💌 Sweet success popup
- 💕 Floating heart animations
- ⚡ Built with pure HTML, CSS, and JavaScript

## 📂 Project Structure

## 🚀 Getting Started

1. Download or clone this repository.
2. Open the project folder.
3. Double-click `index.html` or open it in your browser.
4. Enjoy the interactive experience!
